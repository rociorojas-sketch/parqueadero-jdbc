# Sistema de Gestión de Parqueadero

Proyecto desarrollado en Java utilizando JDBC para la conexión con MySQL.

## Descripción

Este sistema permite gestionar los vehículos que ingresan a un parqueadero, realizando operaciones básicas sobre la base de datos.

## Funcionalidades

- Registrar vehículos
- Consultar vehículos por placa
- Actualizar información de vehículos
- Listar vehículos registrados
- Eliminar vehículos

## Tecnologías utilizadas

- Java
- JDBC
- MySQL
- XAMPP
- NetBeans
- GitHub

## Estructura del proyecto

El proyecto está organizado utilizando una arquitectura por capas:

- **modelo:** clases que representan las entidades
- **dao:** lógica de acceso a datos
- **conexion:** conexión a la base de datos
- **principal:** clase principal del sistema

## Base de datos

La base de datos utilizada se llama:

```
clientes_parqueadero
```

Tabla principal:

```
vehiculo
```
